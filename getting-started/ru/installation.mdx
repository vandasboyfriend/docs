## Установка на macOS

Для установки приложения на устройства под управлением macOS, вам необходимо скачать и запустить установочный файл, который соответствует вашему процессору: Intel или M1/M2

После завершения загрузки запустите скачанный файл **.dmg**, а затем просто переместите значок **Vision** в папку "Программы", как показано на изображении ниже.

<img width="450" className="tailwind mx-auto" src="/images/macos_installer.png" />

После завершения установки вы сможете запустить **Vision** из папки Программы.

## Установка на Linux

Приложение для ОС Linux представлено в двух вариациях: **.AppImage** (образ приложения) и **.deb** (установочный пакет)

Файл с расширением **.AppImage** можно запустить на любой ОС на базе Linux, без каких-либо дополнительных компонентов.

Файл с расширением **.deb** работает как обычный установщик на любой другой ОС. Помимо основного приложения, инсталлер устанавливает необходимые зависимости (библиотеки) для корректной работы.

<Note>
    Дистрибутивы на базе **Debian** (Ubuntu, Linux Mint, Kali Linux и т.д.) поддерживают установку из **.deb** пакета, так что смело можете скачивать именно его.

    Если ваша ОС не поддерживает установку из **.deb** пакета - вам подойдёт образ приложения в формате **.AppImage**
</Note>
