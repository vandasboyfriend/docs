### Заголовки

<ParamField body="X-Token" type="string" required>
  Токен авторизации
</ParamField>

### Параметры запроса

<ParamField body="folder_id" >
  ID удаляемой папки
</ParamField>

<RequestExample>
```js JavaScript (Fetch)
var myHeaders = new Headers();
myHeaders.append("X-Token", "YOUR_TOKEN_HERE");

var requestOptions = {
  method: 'DELETE',
  headers: myHeaders,
  redirect: 'follow'
};

fetch("https://apiv2.empr.cloud/api/v1/folders/:folder_id", requestOptions)
  .then(response => response.text())
  .then(result => console.log(result))
  .catch(error => console.log('error', error));
```

```python Python (Requests)
import requests

url = "https://apiv2.empr.cloud/api/v1/folders/:folder_id"

payload = {}
headers = {
  'X-Token': 'YOUR_TOKEN_HERE'
}

response = requests.request("DELETE", url, headers=headers, data=payload)

print(response.text)

```
</RequestExample>
