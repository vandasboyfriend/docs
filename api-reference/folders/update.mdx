---
title: "Edit folder"
api: "PATCH https://apiv2.empr.cloud/api/v1/folders/:folder_id"
description: "This endpoint updates an existing folder"
---

### Headers

<ParamField body="X-Token" type="string" required>
  your auth token
</ParamField>

### Query parameters
<ParamField body="Folder's id" required>
  your folder's id, you want to change something
</ParamField>


### Body

<ParamField body="folder_name" type="string">
  new folder's name, any string
</ParamField>

<RequestExample>

```js JavaScript Fetch
var myHeaders = new Headers();
myHeaders.append("Content-Type", "application/json");
myHeaders.append("X-Token", "YOUR_TOKEN_HERE");

var raw = JSON.stringify({
  "folder_icon": "Meta"
});

var requestOptions = {
  method: 'PATCH',
  headers: myHeaders,
  body: raw,
  redirect: 'follow'
};

fetch("https://apiv2.empr.cloud/api/v1/folders/b17ba7d5-80f9-4126-aea0-1ce432c4831f", requestOptions)
  .then(response => response.text())
  .then(result => console.log(result))
  .catch(error => console.log('error', error));
```
```python Python (Requests)
import requests
import json

url = "https://apiv2.empr.cloud/api/v1/folders/b17ba7d5-80f9-4126-aea0-1ce432c4831f"

payload = json.dumps({
  "folder_icon": "Moon"
})
headers = {
  'Content-Type': 'application/json',
  'X-Token': 'YOUR_TOKEN_HERE'
}

response = requests.request("PATCH", url, headers=headers, data=payload)

print(response.text)
```
